# 2D Geometry System (Java)

This project implements a simple 2D geometry system using Java and Object-Oriented Programming principles.

## 📌 Project Overview

The system consists of three main classes:

- **Point2D** → Represents a point in a 2D Cartesian coordinate system.
- **Line2D** → Represents a line in slope-intercept form (y = mx + n).
- **Path2D** → Represents a path formed by connecting multiple points.

A `Demo` class is included to demonstrate all implemented methods.

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Encapsulation
- Copy Constructors
- Method Overloading
- Composition
- Basic Geometry Calculations

---

## 📐 Implemented Features

### Point2D
- Distance between two points
- Distance from a point to a line
- String representation

### Line2D
- Parallel check
- Intersection point calculation
- String representation

### Path2D
- Add points
- Calculate total path length
- Combine two paths
- String representation

---

## ▶ How to Run

1. Open the project in any Java IDE.
2. Run the `Demo` class.
3. The console will display the test outputs of all implemented methods.
